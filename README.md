# Rubi.Q
ルービックキューブの展開図を使用した空間認識強化用アプリケーション『Rubi.Q』のソースコードとexeファイルです。

# ファイル構成
Rubi.Q_GameAssets : 使用した画像等の素材が保存されています。 

Rubi.Q_Games : ゲームのソースコード等のUnity6で起動するためのファイルが入っています。 

Rubi.Q_exe : 実行形式のRubi.Qゲーム本体が入っています。

# 各種バージョン等
対応OS : windows11 ,
使用したゲームエンジン : Unity6 ,
プレイ可能人数 : 1人 ,
必要ディスプレイ数 : 2 ,
